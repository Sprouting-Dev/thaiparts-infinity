# 3. หน้าตาระบบหลังเข้าสู่ระบบ

- **แถบบนสุด:** มีคำว่า **Contents** (เนื้อหา) และ **Media** (คลังรูปภาพ)
- **คอลัมน์ซ้าย — “Collections”:** คือกลุ่มเนื้อหาที่แก้ได้ มี 4 กลุ่ม เรียงจากบนลงล่าง:
  1. **สินค้า (Products)**

  3. **ข้อความ Hero หน้าเว็บ**

- **พื้นที่กลาง-ขวา:** จะแสดงรายการของกลุ่มที่เลือกอยู่

> 💡 การใช้งานหลักๆ คือ: **คลิกกลุ่มทางซ้าย → คลิกรายการที่จะแก้ → แก้ → กดเผยแพร่**

# 4. หลักการสำคัญ: แก้ → เผยแพร่ → รอ → ขึ้นเว็บ

อ่านหัวข้อนี้ให้เข้าใจก่อนเริ่มแก้จริง เพราะใช้เหมือนกันทุกหน้า:

1. **แก้ไข** เนื้อหาในช่องต่างๆ
2. กดปุ่ม **“Publish” (เผยแพร่)** มุมขวาบน → เลือก **“Publish now”**
3. **รอประมาณ 2–4 นาที** (ระบบกำลังอัปเดตเว็บไซต์)
4. เปิดเว็บ `thaipartsinfinity.com` แล้ว **กด Refresh (F5)** → จะเห็นการเปลี่ยนแปลง

> ⚠️ **สำคัญ:** หลังกด Publish เว็บจะ **ยังไม่เปลี่ยนทันที** ต้องรอ 2–4 นาทีเสมอ ถ้ายังไม่เห็นให้รอแล้วรีเฟรชอีกครั้ง (อย่ากด Publish ซ้ำๆ รัวๆ)

# 5. การแก้ไข "สินค้า (Products)"

ทำตามทีละขั้น:

1. คลิก **“สินค้า (Products)”** ในคอลัมน์ซ้าย
2. จะเห็นรายการสินค้าทั้งหมด (มีรูป + ชื่อ) → **คลิกที่สินค้าที่ต้องการแก้**
3. จะเข้าหน้าแก้ไข มีช่องต่างๆ **เรียงจากบนลงล่าง** ดังนี้:

   | ลำดับ | ช่อง | คือ / วิธีแก้ |
   |---|---|---|
   | 1 | **ชื่อสินค้า** | ชื่อสั้นของสินค้า (เช่น Bearing & Roller) |
   | 2 | **Slug (URL)** | ⚠️ **อย่าแก้ของสินค้าเดิม** (จะทำให้ลิงก์เสีย) — แก้เฉพาะตอนสร้างใหม่ |
   | 3 | **หัวข้อหลัก** | ชื่อเต็มที่แสดงในหน้าสินค้า |
   | 4 | **แท็ก/หมวด** | หมวดของสินค้า (เช่น Mechanical & Power Transmission Systems) |
   | 5 | **รูปสินค้า** | กดเพื่อเปลี่ยนรูป (ดูวิธีในหัวข้อ 12) |
   | 6 | **รายละเอียด (HTML)** | เนื้อหารายละเอียด (มีตารางสเปกเป็นโค้ด HTML — แก้เฉพาะข้อความ ระวังอย่าลบเครื่องหมาย `<...>`) |
   | 7 | **category** | (ไม่บังคับ — เว้นว่างได้) |
   | 8 | **ลำดับการแสดง** | ตัวเลข เลขน้อยแสดงก่อน |

4. แก้เสร็จ → กด **Publish → Publish now** (มุมขวาบน)
5. รอ 2–4 นาที แล้วเช็กที่หน้า `thaipartsinfinity.com/products`

---

# 6. การเพิ่มสินค้าใหม่

1. คลิก **“สินค้า (Products)”** ในคอลัมน์ซ้าย
2. กดปุ่ม **“New สินค้า”** (มุมขวาบน)
3. กรอกช่องต่างๆ เรียงจากบนลงล่าง (เหมือนตารางในหัวข้อ 5):
   - **ชื่อสินค้า** — ใส่ชื่อ
   - **Slug (URL)** — ใส่ชื่อภาษาอังกฤษพิมพ์เล็ก คั่นด้วยขีด เช่น `ball-valve` (ห้ามมีเว้นวรรค/ภาษาไทย/อักขระพิเศษ)
   - **หัวข้อหลัก** — ชื่อเต็ม
   - **แท็ก/หมวด** — หมวดสินค้า
   - **รูปสินค้า** — อัปโหลดรูป (ดูหัวข้อ 12)
   - **รายละเอียด** — ใส่รายละเอียด
   - **ลำดับการแสดง** — ใส่ตัวเลข (ถ้าไม่แน่ใจใส่เลขมากๆ เช่น 999)
4. กด **Publish → Publish now**
5. รอ 2–4 นาที สินค้าใหม่จะขึ้นในหน้าเว็บ

---

# 7. การแก้ไข "บทความ (Articles)"

1. คลิก **“บทความ (Articles)”** ในคอลัมน์ซ้าย
2. คลิกบทความที่ต้องการแก้
3. ช่องต่างๆ เรียงจากบนลงล่าง:

   | ลำดับ | ช่อง | คือ |
   |---|---|---|
   | 1 | **หัวข้อ** | ชื่อบทความ |
   | 2 | **Slug (URL)** | ⚠️ อย่าแก้ของบทความเดิม |
   | 3 | **หัวข้อรอง** | คำโปรย/คำอธิบายสั้น |
   | 4 | **รูปปก** | รูปหน้าปกบทความ (ดูหัวข้อ 12) |
   | 5 | **เวลาอ่าน (นาที)** | ตัวเลข เช่น 5 |
   | 6 | **วันที่เผยแพร่** | เลือกวันที่ |
   | 7 | **เนื้อหา (เพิ่มทีละบล็อก)** | เนื้อหาบทความ เป็น "บล็อก" ต่อกัน (ดูด้านล่าง) |

   **เรื่องช่อง "เนื้อหา (บล็อก)":**
   - เนื้อหาประกอบด้วยบล็อกหลายอัน เรียงจากบนลงล่าง
   - แต่ละบล็อกมี 2 แบบ:
     - **ย่อหน้าข้อความ** — ใส่ข้อความ (รองรับ HTML)
     - **รูป + คำบรรยาย** — ใส่รูป 1 รูป + คำบรรยายใต้รูป
   - กด **“Add …”** (ปุ่มเพิ่มบล็อก) ด้านล่าง เพื่อเพิ่มบล็อกใหม่ แล้วเลือกแบบที่ต้องการ
   - ลากสลับลำดับบล็อกได้ / กดถังขยะเพื่อลบบล็อก

4. กด **Publish → Publish now** → รอ 2–4 นาที

---

# 8. การเพิ่มบทความใหม่

1. คลิก **“บทความ (Articles)”** → กด **“New บทความ”** (มุมขวาบน)
2. กรอก: หัวข้อ, Slug (อังกฤษพิมพ์เล็ก-ขีด), หัวข้อรอง, รูปปก, เวลาอ่าน, วันที่
3. ที่ช่อง **เนื้อหา** → กดปุ่มเพิ่มบล็อก → เลือก **ย่อหน้าข้อความ** หรือ **รูป + คำบรรยาย** → ใส่เนื้อหา → เพิ่มบล็อกถัดไปได้เรื่อยๆ
4. กด **Publish → Publish now** → รอ 2–4 นาที

---

# 9. การแก้ "ข้อความ Hero หน้าเว็บ"

> "Hero" = แถบใหญ่บนสุดของแต่ละหน้า (รูปพื้นหลัง + ข้อความตัวโต)

1. คลิก **“ข้อความ Hero หน้าเว็บ”** ในคอลัมน์ซ้าย
2. จะมี 3 รายการ: **home** (หน้าแรก), **about-us** (หน้าเกี่ยวกับเรา), **contact-us** (หน้าติดต่อ) → คลิกหน้าที่จะแก้
3. ช่องต่างๆ เรียงจากบนลงล่าง:

   | ลำดับ | ช่อง | คือ |
   |---|---|---|
   | 1 | **ข้อความ Hero (HTML)** | ข้อความตัวใหญ่บน Hero |
   | 2 | **รูปพื้นหลัง Hero** | รูปพื้นหลัง (ดูหัวข้อ 12) |
   | 3 | **ปุ่ม 1 / ปุ่ม 2** | ข้อความบนปุ่ม (ถ้ามี) |
   | 4 | **แสดงปุ่ม** | เปิด/ปิดการแสดงปุ่ม |

4. กด **Publish → Publish now** → รอ 2–4 นาที

> ℹ️ จะไม่เห็นช่อง “SEO” — ระบบเก็บค่า SEO ไว้ให้อัตโนมัติ ไม่ต้องแก้

---

# 10. การแก้ "Footer / ข้อมูลติดต่อ / โลโก้"

> Footer = ส่วนล่างสุดของทุกหน้า (โลโก้ + เบอร์ + อีเมล + ที่อยู่ + โซเชียล)

1. คลิก **“ตั้งค่าเว็บ”** ในคอลัมน์ซ้าย
2. คลิก **“Footer / ข้อมูลติดต่อ / โลโก้”**
3. ช่องต่างๆ เรียงจากบนลงล่าง:

   | ลำดับ | ช่อง | คือ |
   |---|---|---|
   | 1 | **โลโก้** | โลโก้บริษัท |
   | 2 | **รูปแบนเนอร์ก่อน Footer** | รูปแถบใหญ่เหนือ Footer |
   | 3 | **ข้อความก่อน Footer** | ข้อความบนแบนเนอร์ (ขึ้นบรรทัดใหม่ใช้ `<hr>` คั่น) |
   | 4 | **ข้อความปุ่ม** | คำบนปุ่มแบนเนอร์ |
   | 5 | **ชื่อบริษัท** | ชื่อบริษัท |
   | 6 | **ที่อยู่** | ที่อยู่บริษัท |
   | 7 | **เบอร์โทร 1** | เบอร์หลัก (ใส่เฉพาะตัวเลข) |
   | 8 | **เบอร์โทร 2** | เบอร์รอง |
   | 9 | **อีเมล** | อีเมลติดต่อ |
   | 10 | **ลิงก์แผนที่** | ลิงก์ Google Maps |
   | 11 | **โซเชียล** | รายการลิงก์โซเชียล (แต่ละอันมี ประเภท + ลิงก์) |

   **เรื่อง "โซเชียล":** แต่ละรายการมี 2 ช่อง
   - **ประเภท** — พิมพ์ได้: `line`, `facebook`, `email`, `youtube`, `tiktok`, `x`
   - **ลิงก์** — ใส่ URL
   - กด Add เพื่อเพิ่ม / กดถังขยะเพื่อลบ

4. กด **Publish → Publish now** → รอ 2–4 นาที

# 11. การแก้ "หน้าเกี่ยวกับเรา"

1. คลิก **“ตั้งค่าเว็บ”** ในคอลัมน์ซ้าย
2. คลิก **“หน้าเกี่ยวกับเรา”**
3. ช่องต่างๆ เรียงจากบนลงล่าง (แต่ละหัวข้อมี **หัวข้อ** และ **เนื้อหา**):

   | ลำดับ | ช่อง | คือ |
   |---|---|---|
   | 1 | **เกี่ยวกับ** | หัวข้อ + เนื้อหา ส่วนแนะนำบริษัท |
   | 2 | **วิสัยทัศน์** | หัวข้อ + เนื้อหา วิสัยทัศน์ |
   | 3 | **พันธกิจ** | หัวข้อ + เนื้อหา พันธกิจ |
   | 4 | **ทีมงาน** | รูป + คำบรรยาย |
   | 5 | **คลังสินค้า** | รูป + คำบรรยาย |
   | 6 | **โลโก้มาตรฐาน (รูป)** | รายการรูปโลโก้มาตรฐาน (เพิ่ม/ลบได้) |

4. กด **Publish → Publish now** → รอ 2–4 นาที

# 12. วิธีอัปโหลด/เปลี่ยนรูปภาพ

ทุกที่ที่มีช่อง **“รูป”** ใช้วิธีเดียวกัน:

1. คลิกที่ช่องรูป (หรือปุ่ม Choose / รูปเดิม)
2. จะมีหน้าต่าง **“Media assets”** เปิดขึ้น
3. มี 2 ทางเลือก:
   - **อัปโหลดรูปใหม่:** กดปุ่ม **“Upload”** มุมขวาบน → เลือกไฟล์รูปจากเครื่อง → รอสักครู่ → คลิกรูปที่อัป → กด **“Choose selected”**
   - **ใช้รูปที่เคยอัปแล้ว:** คลิกรูปในคลัง → กด **“Choose selected”**
4. รูปจะมาแสดงในช่อง → กด **Publish** ตามปกติ

> 💡 หน้าต่าง Media ตอนแรกอาจว่าง (ขึ้น “No assets found”) เป็นเรื่องปกติ เพราะรูปเดิมของเว็บเก็บไว้คนละที่ — พออัปรูปใหม่ รูปจะมาอยู่ในคลังนี้
>
> 💡 **ขนาดรูปแนะนำ:** ย่อรูปให้ไม่เกิน ~1–2 MB ต่อไฟล์ (รูปใหญ่เกินไปจะทำให้เว็บโหลดช้า)

# 13. ข้อควรระวัง (อ่านให้ครบ)

- ⚠️ **อย่าแก้ช่อง “Slug (URL)” ของรายการเดิม** — จะทำให้ลิงก์หน้านั้นเสีย และ Google จะหาไม่เจอ (แก้ Slug ได้เฉพาะตอนสร้างรายการ **ใหม่** เท่านั้น)
- ⚠️ **หลัง Publish ต้องรอ 2–4 นาที** เว็บถึงจะเปลี่ยน — อย่ากด Publish ซ้ำรัวๆ
- ⚠️ ในช่องที่เป็น **HTML** (รายละเอียดสินค้า/ตารางสเปก) ให้แก้เฉพาะ "ข้อความ" ระวังอย่าลบเครื่องหมาย `<` `>` หรือ `</table>` เพราะจะทำให้หน้าเพี้ยน
- ⚠️ **อย่าลบ** รายการ/บล็อก ถ้าไม่แน่ใจ — ลบแล้วต้องสร้างใหม่
- ✅ ถ้าแก้ผิดแล้วยังไม่ได้กด Publish → กดออก (ไม่บันทึก) ได้ ข้อมูลเดิมจะไม่เปลี่ยน
- ✅ รหัสเข้าระบบ **เก็บเป็นความลับ** อย่าให้คนอื่น

# 14. ปัญหาที่พบบ่อย และวิธีแก้

**Q: แก้แล้วกด Publish แล้ว แต่เว็บยังไม่เปลี่ยน?**
A: รอให้ครบ 2–4 นาทีก่อน แล้วกด Refresh (F5) ที่หน้าเว็บ / ลองกด Ctrl+Shift+R เพื่อล้างแคช

**Q: เข้า /admin แล้วขึ้นหน้า Login ทุกครั้ง?**
A: ปกติระบบจะจำ login ให้ ถ้าขึ้นบ่อยอาจเป็นเพราะเบราว์เซอร์ล้าง cookie — login ใหม่ตามหัวข้อ 2 ได้เลย

**Q: ลืมรหัสผ่าน?**
A: ติดต่อทีมงานเพื่อขอรีเซ็ตรหัสใหม่ (ทีมงานตั้งให้ได้)

**Q: อัปรูปแล้วรูปไม่ขึ้น?**
A: เช็กว่ากด **“Choose selected”** หลังเลือกรูปแล้วหรือยัง / รูปต้องเป็นไฟล์ภาพ (.jpg .png .webp)

**Q: อยากแก้หน้า "บริการ (Services)" หรือสินค้าที่โชว์หน้าแรก?**
A: ส่วนนี้ทีมงานดูแลให้ (ซับซ้อน) — แจ้งทีมงานได้เลย
